Metodos de Arrays:

Abajo va un listado de los metodos que voy usando en un array:

  • array.indexOf(valorBuscar, fromIndex) - recorre todo el array y retorna EL PRIMER INDICE del valor de un elemento que coincide restritamente (===) con valorBuscar.
  • array.forEach( function callback(value, valueIndex, array), thisValue) => { }) - itera, o sea ejecuta la funcion para cada item del array, aplicando los parametros value, valueIndex, y array.
