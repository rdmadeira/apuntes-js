Funciones:

Las funciones son definidas a partir de 2 formas: por declaración con la palabra function, y con una expresión de función.

  • Declaración de función - function funcName (param1, param2, etc.) { instrucciones javaScript }
  • Expresión de función - let/const variableName = function (param1, param2, etc.) { instrucciones javaScript}. Puedes colocar un nombre de función, si necesitás incluirla adentro de si misma. let variable = function funcName (param1, param2, etc.) { instrucciones javaScript llamando a si misma.}. Es este método, no se aplica la caracteristica de hoisting de llamado de función.

Las funciones para funcionaren, necesitan ser llamadas o sea ejecutadas. Pueden ser llamadas en el mismo scope en que fue declarada, o puede ser ejecutada como parametro de otra función o inclusive adentro de si misma. Son las llamadas funciones recursivas.

Las funciones recursivas funcionan como un bucle (loop). Para evitar su ejecución infinita, hay una condición que la limita.

Ambito de una función:

Una variable declarada adentro de una función no puede ser accedida afuera de la misma. Esto se llama scope (ámbito) de esta función. Lo importante es que una función sí puede acceder a una variable declarada en el ámbito de esta función. En otras palabras, una función definida en el ámbito global puede acceder a todas las variables definidas en el ámbito global. Una función definida dentro de otra función también puede acceder a todas las variables definidas en su función principal y a cualquier otra variable a la que tenga acceso la función principal.

Return - el return se usa adentro de las instrucciones de la función. El mismo devuelve el valor final a la llamada de esta función. MUY IMPORTANTE!! Al aplicar return, no se leé las instrucciones que vienen después del mismo, es como se termina la función.
