Variables

Las variables son "conteinedores" de valores;

Las mayusculas importan para las variables, asi que es distinto una variable ej.: var a = content =! var A = content

Si una variable no está inicializada todavía, su valor es undefined.

Scope o ambito de una variable - es la zona en la cual una determinada variable y su valor existen.

console.log(e); // undefined;

var e = 1;

console.log(e); // 1

En el ejemplo anterior, el ámbito de la variable "e" comienza a partir de su inicialización y "vive" hasta el final del programa. Por esta razón, el valor que muestra el primer console.log() es undefined. A esto se le llama ámbito global y es el ejemplo más sencillo. Veamos otro ejemplo:

Tenemos en este caso el ambito global con el valor de a siendo 10 y el ambito local de la función x siendo su valor 5. Un ambito no interfiere en el otro, si el mismo nombre de variable es declarado adentro y afuera de una función.

El objeto window nos permite acceder al valor de la variable "a" en el ámbito global. Esto ocurre así porque las variables globales se almacenan dentro del objeto window.

Reglas básicas de nombres de variables:

Los nombres de las variables solo pueden ser iniciadas con una letra mayuscula o minuscula o guion bajo (_).

No se pueden dejar espacios en los nombres de las variables.

Tipos de datos (valores) de las variables:

Number (numero), string (cadena de caracteres de texto), booleano (true or false), function (funcion), undefined (no inicializada), object, null (no existe)

javaScript es un lenguaje dinamico porque identifica el tipo de dato de acuerdo con el valor de la variable automaticamente. Y por la misma variable, se puede cambiar de tipo de datos solo en cambiar su valor.

Para identificar que tipo de datos tiene una variable, ya que muchas veces está implícito por una función o otra cosa, basta usar la función typeof en console.log.

La conastante NaN - es un valor especial en js que significa "not a number" y puede ser el resultado de operaciones matemáticas inposibles o indeterminados

  • 0 / 0; - indeterminado
  • 4 + "Rodrigo"; - imposible
  • Nan + 4 - con NaN operando

Para comprobar si una variable tiene valor NaN o sea, ha fallado o da algo indeterminado, basta usar: Number.isNaN(e); // true or false

El valor NaN es definido como numerico. En otras palabras, dentro de los tipos de datos numéricos, NaN es un conjunto de números que no se pueden representar.

Resumiendo, para consultar que naturaleza tiene una variable numerica, tenemos las siguientes funciones:

  • Number.isFinite(n) - Comproba si un numero es finito (booleano)
  • Number.isInteger(n) - Comproba si un numero es entero (booleano)
  • Number.isSafeInteger(n) - Comproba si un numero es seguro (booleano)
  • Number.isNaN(n) - Comproba si no es un numero (booleano)
