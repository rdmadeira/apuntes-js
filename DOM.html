HTMLCollection y NodeList:

Son 'arraylike', simulaciones de array, conteniendo que contienen propriedad length y puedese acceder a un elemento con el indice entre corchetes, igual a los Arrays.

    HTMLCollection:
  • Contiene los elementos de etiqueta del document.
  • No permite uso del metodo de array forEach.
  • No contiene nodes entre sus elementos.
  • Es el resultado de metodos como getElementsByTagName, getElementsByClassName.
    NodeList:
  • Contiene los nodos de etiqueta y los subnodos, como atributos, textos, comentarios, quiebras de linea.
  • Permite metodos como forEach.
  • Es el resultado de metodos como getElementsByName, querySelectorAll.

Metodos y propriedades de elementos para manipular Atributos:

  • elemento.setAttribute('nombre-de-atributo', 'valor-del-atributo').
  • elemento.getAttribute('nombre-de-atributo') - devuelve el valor del atributo.
  • elemento.classList - devuelve un DOMTokenList, que es un array-like, con muchos metodos.
  • elemento.classList.contains('un-valor-de-clase')
  • elemento.classList.add('un-valor-de-clase') - agrega un valor a la clase
  • elemento.classList.remove('un-valor-de-clase') - borra el valor de la clase especificada
  • elemento.classList.toggle('un-valor-de-clase')
