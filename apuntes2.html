Tipos de Datos - Arrays

Arrays son conjunto o grupo de datos que tienen alguna relación entre si que son declarados en una variable.

Un array puede ser mixto, o sea tener datos de distintas naturalezas (number, string, booleanos, etc.).

Hay 2 formas de declarar una variable con valor siendo un array:

  • Metodo literal: Entre corchetes [] , y cada elemento separado por commas - Ej.: var a = ["a", 50, true]; - Metodo recomendado
  • Como un objeto: var a = new Array("a", 50, true);

Propriedades del Array:

.lenght - parecido con strings, pero esta propriedad devuelve la cantidad de elementos que contiene.

array[num] - accede al valor del elemento que está posicionado en la posición num. Los arrays tiene como su primer elemento en la posición 0. Si el valor de num es superior a la cantidad de elementos, su valor devuelto será undefined.

Funciones para añadir o eliminar elementos:

  • array.push(valor0, valor1, ...) - agrega valor0, valor1,... después del ultimo elemento del array y devuelve su nuevo lenght;
  • array.pop() - elimina el ultimo elemento del array y devuelve el valor de este elemento eliminado;
  • array.unshift(valor0 ,valor1, ...) - agrega valor0, valor1, ... al inicio del array y devuelve su nuevo lenght;
  • array.shift() - elimina el primer elemento del array y devuelve el valor de este elemento eliminado.
  • array.concat(valor0, valor1, ...) - Devuelve el valor del array agregando los valores del concat al final, pero sin modificar el array. A parte de eso, tiene una diferencia importante al ingresar otro array de valores, vea el segundo ejemplo.

Metodos de creación de arrays con base en otras variables o objectos:

  • array.slice(ini, end) - devuelve los valores entre array[ini] y array[end], incluindo ini y EXCLUINDO end. Es bueno para crear arrays mas pequeños basado en otro mas grande;
  • array.splice(ini, num) - elimina y devuelve num valores empezando con el ini. A la diferencia de slice(), el splice() cambia el array inicial. El num es el tamaño del array eliminado y creado.
  • array.join(string) - Devuelve un string concatenando los valores de un array, separados por el string entre los parentesis.
  • split(string) - devuelve un array, eliminando el string entre parentesis de un string más grande que contiene este string a ser eliminado entre otros valores. Es el contrario de join./li>
